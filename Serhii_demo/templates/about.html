- App is a json fileshare. You can upload json files to general storage using 'Upload File to Storage' button from home page. You can upload files which contain only valid json data and have 'json' extension.

- To look through all files from general storage, delete, update or download them you can use 'View Stored Files' button.

- Using 'Register' button you can create new unique user.

- After registration you can login, upload, view, update and delete files from your personal storage.

- After login, using 'Account' button you can update your username and account image.
